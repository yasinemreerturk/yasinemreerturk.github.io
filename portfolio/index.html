AI Post Maker

Kişisel full-stack proje

  • Asenkron görevler için görsel üretim araçları ve arka plan işleme süreçleri entegre edilmiştir.

Teknolojiler: Ruby on Rails, PostgreSQL, Redis, Sidekiq, Turbo/Hotwire, Docker, REST APIs, Background Jobs, Basic DevOps Setup
