# Multi-Page Developer Portfolio (Static)

Dark engineering-style portfolio built as static multi-page HTML/CSS/JS with Tailwind via CDN.

## Pages

- `/` -> Home
- `/about/` -> About
- `/portfolio/` -> Portfolio
- `/contact/` -> Contact

## Stack

- HTML5 + semantic sections
- TailwindCSS CDN config (inline per page)
- Shared custom styles: `assets/style.css`
- Shared behavior (mobile nav, active link, form message): `assets/script.js`

## Local Run

Open `index.html` in a browser.

## GitHub Pages

1. Go to `Settings` -> `Pages`
2. Source: `Deploy from a branch`
3. Branch: `main`
4. Folder: `/(root)`
